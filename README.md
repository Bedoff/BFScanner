# BFScanner
GO based port scanner (under development)

install
-------

$ apt install python
$ apt install git
$ apt install go
$ cd Desktop
$ git clone https://github.com/Bedoff/BFScanner
$ cd BFScanner
$ python3 bfscanner.py


some code you can try if you get an error

$ sysctl -w vm.overcommit_memory=1
$ ulimit -n 50000
